# Mobile App Development Environment Variable Manager VS Code Extension

> View, edit, and sync .env files across environments with encryption support. Never deploy with wrong environment vars again.

## Features

- Core functionality

## Installation

### From VS Code Marketplace
1. Open VS Code
2. Press `Ctrl+P` (or `Cmd+P` on Mac)
3. Type `ext install digitalforge.62676919`
4. Press Enter

### From VSIX file
1. Download the `.vsix` file
2. In VS Code: `Extensions > ··· > Install from VSIX`
3. Select the downloaded file

## Usage

Press `Ctrl+Shift+P` and search for `Mobile App Development Environment Variable Manager VS Code Extension` to see all available commands.

**Keyboard shortcut:** `Ctrl+Shift+Alt+A` (or `Cmd+Shift+Alt+A` on Mac)

## Settings

| Setting | Default | Description |
|---------|---------|-------------|
| `62676919.enabled` | `true` | Enable/disable the extension |
| `62676919.autoRun` | `false` | Auto-run on file open |
| `62676919.logLevel` | `"info"` | Log verbosity |

## Requirements

- VS Code 1.74.0 or higher
